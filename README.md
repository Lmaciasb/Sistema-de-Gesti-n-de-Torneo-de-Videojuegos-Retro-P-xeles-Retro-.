# Sistema-de-Gesti-n-de-Torneo-de-Videojuegos-Retro-P-xeles-Retro-.
Motor en Python para e-sports. Simula un entorno transaccional real: ingesta de datos, persistencia (JSON), emparejamiento algorítmico y motor estadístico. Calcula el rendimiento y genera reportes de Ranking automatizados. Evidencia sólida de lógica de programación y modelado de datos.
