# Sistema de Gestion de Torneo de Videojuegos Retro.
Motor en Python para e-sports. Simula un entorno transaccional real. Que permite el ingreso de datos, persistencia (JSON), emparejamiento algorítmico y motor estadístico. Calcula el rendimiento y genera reportes de Ranking automatizados. Evidencia sólida de lógica de programación y modelado de datos.
Automatiza el matchmaking, transforma resultados brutos en KPIs estratégicos (% Efectividad), generando un Ranking dinámico. Demuestra arquitectura modular, limpieza de código y enfoque UX.
El sistema fue pensado primeramente con nombres de Equipos de F1, pero complemetables con nombres de equipos aletoriamente asignados por un color
